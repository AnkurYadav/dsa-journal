# LC 15 - 3sum

**Difficulty:** Medium
**Pattern:** two pointers
**Link:** https://leetcode.com/problems/3sum

## Approach

for each index i find the pair j and k greater than i using two pointers approach such that arr[j]+arr[k]=-arr[i]

## Code

```python
class Solution:
    def threeSum(self, nums: list[int]) -> list[list[int]]:
        l = len(nums)
        if l < 3:
            return []

        sorted_nums = sorted(nums)
        if sorted_nums[0] > 0:
            return []
        elif sorted_nums[0] == 0 and sorted_nums[2] != 0:
            return []

        res = []

        for i in range(l-2):
            if i > 0 and sorted_nums[i] == sorted_nums[i-1]:
                continue
            target = -sorted_nums[i]
            j = i+1
            k = l-1
            while j<k:
                sum = sorted_nums[j] + sorted_nums[k]
                if sum < target:
                    j+=1
                elif sum > target:
                    k-=1
                else:
                    res.append([sorted_nums[i], sorted_nums[j], sorted_nums[k]])
                    j+=1
                    while sorted_nums[j] == sorted_nums[j-1] and j < k:
                        j+=1
        
        return res
```

## Complexity

- Time: O(n^2)
- Space: O(n) if using separate array to sort else if using same array O(1)

## Gotcha

keep in mind to handle duplicates using checks for repeated numbers
